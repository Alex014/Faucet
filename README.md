# Faucet
Crypto faucet based on time-tokens (proof of time)  

Currently supported only Privateness (NESS) crypto, but it can be easily rewriten for any other crypto.

# Install
## Requirements
 * PHP 7.3 +
 * php-sodium
 * php-sqlite3
